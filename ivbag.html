IV Bag Monitoring System

Project Overview

The IV Bag Monitoring System is an IoT-based healthcare solution designed to improve patient safety by automating IV fluid tracking. Using an ESP32 with HX711 and a load cell, the system continuously measures fluid levels and displays them locally on an LCD screen. A buzzer alert is triggered when the IV bag drops below a set threshold, ensuring timely intervention. Real-time data is synced to Firebase, enabling a Flutter mobile app to provide remote monitoring, notifications, and logs for healthcare staff. This solution minimizes the risk of delayed IV bag replacement, enhances hospital efficiency, and supports better patient care through smart monitoring.

Tech Stack Used

Hardware

ESP32 microcontroller, HX711 amplifier, Load cell (for weight measurement)

Mobile App

Flutter framework (remote monitoring, notifications, logs)

Backend & Database

Firebase Realtime Database (data syncing, storage, alerts)

IoT Communication

Wi-Fi connectivity via ESP32

Peripherals

LCD display (local fluid level display), Buzzer (low-level alert system)

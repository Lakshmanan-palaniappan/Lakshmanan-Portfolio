Awesome Notes

Project Overview

A mobile application built using Flutter and Firebase to help users create, organize, and manage personal notes efficiently. The app incorporates Google Sign-In for secure authentication and uses the Provider package for state management, ensuring smooth performance. Users can add, edit, delete, and categorize notes while data is stored in Firebase for real-time synchronization across devices. This project emphasizes mobile UX, cloud integration, and responsive design.

Tech Stack Used

Frontend

Flutter Widgets (UI components, responsive design)

Framework

Flutter

Backend & Database

Firebase (Realtime Database and Authentication)

Authentication

Google Sign-In for secure login

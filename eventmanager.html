Event Manager

Project Overview

The Event Manager is a web-based application developed using Django with SQL Server (SSMS) as the backend to simplify event organization and participation. The system provides administrators with features to create, update, and remove events while tracking participant registrations in real time. Users can register for events through their email, receive automated confirmation messages, and get notified of event updates. The project includes well-structured database schemas and optimized queries to efficiently manage events and participants, ensuring scalability and reliability. This solution streamlines event handling, improves communication, and enhances the overall event management experience.

Tech Stack Used

Framework

Django (Python-based web framework)

Frontend

Django Templates, HTML, CSS, JavaScript

Backend & Database

SQL Server (SSMS)

Authentication & Notification

Email confirmation, automated event update notifications

Peripherals

LCD display (local fluid level display), Buzzer (low-level alert system)

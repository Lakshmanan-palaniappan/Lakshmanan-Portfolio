Autocart - The Future of Shopping

Project Overview

Autocart is an autonomous shopping cart system designed to transform the retail experience by combining computer vision, IoT, and mobile technology. The cart is equipped with YOLOv4 object detection and a Raspberry Pi for real-time product recognition, autonomous navigation, and obstacle avoidance. An Arduino UNO manages sensor integration, while Firebase enables cloud connectivity and transaction handling. A Flutter-based mobile app provides customers with digital billing, easy payments, and a live view of their scanned products. By automating checkout and reducing manual intervention, Autocart enhances store efficiency, minimizes wait times, and creates a seamless, tech-driven shopping experience.

Tech Stack Used

Hardware

Raspberry Pi (real-time processing, navigation, obstacle avoidance), Arduino UNO (sensor integration)

Computer Vision

YOLOv4 (object detection for product recognition)

Backend & Cloud

Firebase (cloud connectivity, data storage, transaction handling)

IoT & Sensors

Ultrasonic/infrared sensors for navigation and obstacle detection

Mobile App

Flutter framework (digital billing, live product view, payment handling)

Integration

Real-time synchronization between hardware, mobile app, and cloud services

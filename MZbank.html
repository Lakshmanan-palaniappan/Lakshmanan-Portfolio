MZ Bank

Project Overview

A desktop ATM simulation application developed using Python and Tkinter with SQL Server (SSMS) as the backend. The system allows users to create accounts, deposit and withdraw money, and manage their account securely with an ATM PIN. The project demonstrates handling of database transactions, user authentication, and GUI-based interaction for banking operations. This project highlights backend database integration with a desktop interface for financial transactions.

Tech Stack Used

Framework

Python Tkinter (GUI development)

Frontend

Tkinter GUI components

Backend & Database

SQL Server (SSMS)

Authentication

ATM PIN-based user login and verification
